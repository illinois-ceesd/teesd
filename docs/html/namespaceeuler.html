Computes the inviscid fluxes in one dimension.

This routine computes the inviscid fluxes of the single gas NS in one dimension. For example in the direction, the inviscid flux is:

Parameters
numDim- const integer indicating number of dimensions
numPoints- const 64-bit integer indicating total size of input arrays
gridSizes- const 64-bit integer 1d array of size numDim, which indicates the number of points in each dimension for the grid
opInterval- const 64-bit integer 1d array of size 2 x numDim which indicates the rectangular interval over which this routine should operate. e.g. [iStart iEnd jStart jEnd ... ]
fluxDir- const integer indicating the direction in which the flux is to be calculated
gridType- const integer indicating whether the gridMetric parameter is a Cartesian, Rectilinear, or Curvilinear metric
gridMetric- const double precision 1d array containing the grid metric
rhoBuffer- ( ) const double precision contiguous 1d array of size numPoints representing the density at each point on the grid
rhoVBuffer- ( ) const double precision contiguous 1d array of size numDim x numPoints representing the momentum density at each point on the grid.
rhoEBuffer- ( ) const double precision contiguous 1d array of size numPoints representing the energy density at each point on the grid
velHat- ( ) - const double precision contiguous 1d array of size numPoints representing the component of the contravariant velocity corresponding to the direction (fluxDir) in which the fluxes are to be calculated
pressureBuffer- (p) - const double precision contiguous 1d array of size numPoints representing the pressure at each point on the grid
fluxBuffer- double precision contiguous 1d array of size (numDim + 2) x numPoints for output of the fluxes for each equation. The fluxBuffer is returned with (numDim + 2) blocks of size numPoints, one block for each equation.
