Typical Usage

Typical usage examples:

  • Start PlasCom2 from scratch using built-in grid generators:

mpiexec -n <numproc> plascom2x -v 2 -p 2 -c <configfile>

Note
The "-v 2" and "-p 2" options are optional and set the verbosity and profiling levels, repsectively
  • Restart PlasCom2 with no configuration changes from initial run:

mpiexec -n <numproc> plascom2x -v 2 -p 2 <restartfile> <targetfile>

  • Restart PlasCom2 with changes to configuration:

mpiexec -n <numproc> plascom2x -v 2 -p 2 -c <configfile> <restartfile> <targetfile>

  • Restart PlasCom2 from a PlasComCM input deck:

mpiexec -n <numproc> plascom2x -v 2 -p 2 -c <plascomcm.inp> <PlasComCM *.h5 restart file> <PlasComCM *.h5 target file>

Note
PlasCom2 will create a (mostly) PlasCom2-compatible configuration file and a solution dump. The generated config file needs to be stripped of PlasComCM parameters before used directly in PlasCom2.

Configuration File

PlasCom2 is configured with a configuration file, analagous to the legacy 

plascomcm.inp

. Please see below for documentation on specific parameters by section.

PlasCom2 Section

The PlasCom2 parameter section controls the overall system-level stepping, and major run modes.

  • PlasCom2:NumSteps - max number of steps in the simulation
  • PlasCom2:NumStepsIO - number of steps between solution dumps
  • PlasCom2:NumStepsStatus - number of steps between status reports
  • PlasCom2:RunMode - indicates the major running mode of PlasCom2
    • RunMode = PC2 - runs PlasCom2 in the usual "simulation" mode
    • RunMode = Maxwell - runs PlasCom2 in Maxwell solver mode
    • RunMode = BareBones - just makes sure PlasCom2 can run - no simulation is started

Geometry Section

The Geometry configuration section specifies the settings for the geometries and grids used in the simulation. Each so-called geometry is just a collection of grids.

  • Geometry:Names = <geomName1> <geomName2> <geomName3> - space delimited list of geometry names
  • geometryName:GridNames = <gridName1> <gridName2> <gridName3> - space delimited list of grids for geometry *<geometryName>*
  • geometryName:gridName:Dimension = 2 | 3 - specifies spatial dimensionality of grid
  • geometryName:gridName:NumPoints = nX nY [nZ] - number of points in each dimension of the grid
  • geometryName:gridName:Type = Uniform | Rectilinear | Curvilinear - chooses the type of grid metric used for this grid
  • geometryName:gridName:PhysicalExtent = xStart xEnd yStart yEnd [zStart zEnd] - for uniform grids, the extents of the physical coordinates
