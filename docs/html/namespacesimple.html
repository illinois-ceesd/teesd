ZAWPXY kernel performs Z = aW + XY with scalar a, contiguous input vectors W, X, and Y, and output vector Z. Operand arrays are contiguous of size numPoints. The kernel operates on the rectangular interval specified by bufferInterval. The shape of the input and output arrays are specified by bufferSize, which is an numDim - dimensional array that specifies the size in each of numDim dimesions.

Parameters

bufferSize- const 64-bit integer array of size numDim specifies the size of the input and output arrays in each of numDim dimensions
bufferInterval- const 64-bit integer array of size 2 x numDim indicating the rectangular interval in which the kernel should operate; e.g. [ iStart iEnd jStart jEnd ]

ZXY point-wise operator performing Z = XY (all vectors)

ZXY performs Z = XY where X, Y are each contiguous arrays. Operand arrays are contiguous of size numPoints. The kernel operates on the rectangular interval specified by bufferInterval. The shape of the input and output arrays are specified by bufferSize, which is an numDim - dimensional array that specifies the size in each of numDim dimesions.

Parameters
numDim- const integer input specifies the number of dimensions of the input and output arrays
numPoints- const 64-bit integer input specifies the total size of the input and output arrays
bufferSize- const 64-bit integer array of size numDim specifies the size of the input and output arrays in each of numDim dimensions
bufferInterval- const 64-bit integer array of size 2 x numDim indicating the rectangular interval in which the kernel should operate; e.g. [ iStart iEnd jStart jEnd ]
X- const double precision input array
Y- const double precision input array
Z- double precision output array
