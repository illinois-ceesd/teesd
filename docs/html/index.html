PlasCom2 Documentation

Introduction

PlasCom2 is XPACC's next-generation multiphysics simulation application, developed as a replacement for the current flagship, PlasComCM. PlasCom2 uses the IX (Illinois Interface to X) toolkit and uses the following internal and third-party packages:

  • CMake
  • Doxygen
  • IX
  • MPI
  • HDF

PlasCom2 also interfaces to many of XPACC's Computer Science tools through the CS-Tool Unification Framework (CSTUF). Online documentation for CSTUF can be located here.
